Interview Questions & Answers

1. What is the difference between JDK, JRE, and JVM?

Answer: JDK (Java Development Kit) is a software development kit with tools like javac for compiling Java code. JRE (Java Runtime Environment) provides libraries and JVM to run Java applications. JVM (Java Virtual Machine) interprets compiled Java bytecode to execute programs.

2. What is the significance of the main method in Java?

Answer: The main method is the entry point of a Java application, with the signature public static void main(String[] args). It is called by the JVM to start program execution.

41. What are the four pillars of OOP in Java?

Answer: Encapsulation, Inheritance, Polymorphism, and Abstraction.

42. What is encapsulation in Java?

Answer: Encapsulation bundles data (fields) and methods into a class, restricting direct access to fields using access modifiers (e.g., private) and providing public getters/setters.

66. What is an ArrayList in Java?

Answer: ArrayList is a resizable, dynamic array implementation of the List interface, allowing random access and modification.

67. What is a HashMap in Java?

Answer: HashMap is a hash table-based implementation of the Map interface, storing key-value pairs with fast lookup.

96. What is an exception in Java?

Answer: An exception is an event that disrupts normal program execution, such as NullPointerException or IOException.

97. What is the purpose of the try-catch block?

Answer: The try block contains code that might throw an exception. The catch block handles specific exceptions thrown in the try block.

111. What is an RDBMS?

Answer: An RDBMS (Relational Database Management System) stores data in tables with rows and columns, enforcing relationships via keys.

112. What is a primary key?

Answer: A primary key uniquely identifies each record in a table and cannot be null.
